
# weBudget - The Collaborative Budget System
#### The purpose of our application is to assist in managing group/roommate related expenses using shared wallets.
###### to run our project simply clone our repo, open the "weBudget" directory that you will receieve after cloning in android studio and select build/run which will build and launch the application.
